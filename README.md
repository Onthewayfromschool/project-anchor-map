# 项目锚点地图生成器

快速生成前端/全栈项目的代码定位地图,帮你从页面 UI 直接跳转到对应代码位置。

## 用途

当你面对一个项目,想快速搞清楚:
- 这个页面的代码在哪个文件?
- 这个按钮点击后跳转到哪里?
- 页面上这块区域对应哪个组件?

这个 skill 会帮你生成一份速查地图,不用在代码里到处翻。

## 触发方式

对 AI 说这些话就能触发:
- "帮我梳理项目结构"
- "生成项目地图"
- "我想快速定位代码"
- "项目导航"

## 生成内容

1. **路由总览表** - 所有页面 URL 和对应文件
2. **页面跳转关系图** - 页面之间怎么跳转的
3. **页面区域地图** - 每个页面的 UI 布局和组件位置(用 ASCII 图展示)
4. **State 管理速查** - 全局状态存在哪些文件里
5. **定位套路指引** - 遇到 bug 怎么快速找到代码

## 特点

- **简洁优先** - 表格 + ASCII 图,不写长篇文档
- **从用户视角出发** - 按"页面上看到什么"组织,不是按代码目录
- **直接定位** - 每个区域都标注具体文件路径和行号

## 适用项目

React、Vue、Next.js、Remix 等主流前端框架的项目都可以用。
